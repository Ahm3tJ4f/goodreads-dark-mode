# Goodreads Dark Mode

finally. dark mode for goodreads.

## what it does

turns the goodreads website dark. colors are hand-picked to be easy on the eyes. no blinding white backgrounds at 2am.

## what might be broken

some elements might still be bright. i probably missed a few classes. if something looks off, open an issue or fix it yourself and send a pr.

## installation

1. go to `chrome://extensions`
2. turn on **developer mode** (top right corner)
3. click **load unpacked**
4. select the folder

or grab it from the chrome web store if i ever publish it there.

## what's next (maybe)

found something broken? fix it. open a pr. or don't. i'm not your manager.
